1. 基础文件管理模块 (Core File System)
文件操作

单文件/多文件上传（支持拖拽）

文件夹创建、重命名、移动、删除

回收站功能（彻底删除与一键还原）

浏览与搜索

列表/网格视图切换

按名称、大小、修改时间排序

全局实时搜索（支持文件名模糊匹配）

高级交互

右键快捷菜单

文件/文件夹属性查看

2. 高级传输引擎 (High-Speed Transfer)
上传优化

秒传功能： 客户端计算 Hash，服务端匹配去重

分片上传： 大文件切片并行上传

断点续传： 记录已上传 Block，网络中断后自动恢复

下载优化

多线程加速下载

多文件批量打包下载 (.zip)

3. 文档版本与协作 (Version & Collaboration)
版本控制 (Git-Like)

历史版本自动记录（每次 Commit 产生新版本）

版本预览与差异对比

一键回滚至历史版本

多人协作

资料库共享： 支持将整个 Repo 共享给其他用户/团队

权限分级： 预览者、查看者、下载者、管理者

实时同步： 客户端检测服务端变更并自动拉取

4. 在线办公与预览 (Online Office)
多媒体预览

图片（JPG/PNG/WebP）缩略图与大图查看

视频/音频流式播放

PDF 在线阅读

文档编辑

Markdown 编辑器（支持实时预览）

代码文件高亮查看 (Go, Python, JS, etc.)

(进阶) Office 文档在线协作 (集成 OnlyOffice/LibreOffice)

5. 系统安全与管理 (Security & Admin)
账户安全

JWT 身份认证

两步验证 (2FA/OTP)

登录日志与操作审计

数据安全

端到端加密： 资料库级独立加密（服务端无法读取内容）

链接分享： 支持设置提取码、到期时间、访问次数限制

后台管理

用户空间配额限制

全局存储状态监控

系统参数配置
