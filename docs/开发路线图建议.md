场景 A：初始化后端存储引擎
Prompt: > “请作为高级 Go 工程师，为 Sealock Doc 编写后端核心存储层的原型。

定义 Block 和 File 的结构体。

实现一个文件切块函数：将文件按照 2MB 大小切块，计算每块的 SHA-256 Hash 值。

支持拖拽上传。
